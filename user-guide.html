Interactive Bloch Sphere Visualizer

User Guide for learners, educators, and researchers

1. Introduction

The Interactive Bloch Sphere Visualizer is designed to help learners, educators, and researchers understand single-qubit quantum states using a geometric Bloch sphere representation.

2. Bloch Sphere Basics

  • +Z axis → |0⟩
  • −Z axis → |1⟩
  • +X / −X → |+⟩ / |−⟩
  • +Y / −Y → |+i⟩ / |−i⟩

3. Mouse Interaction

Click and drag the Bloch sphere to rotate the view. This changes only the visualization, not the quantum state.

Rotating the sphere with the mouse does NOT change θ or φ.

4. Polar and Azimuthal Angle Sliders

The sliders control the qubit state according to:

|ψ⟩ = cos(θ/2)|0⟩ + eiφ sin(θ/2)|1⟩

5. State Information Panels

  • Angles (θ, φ)
  • Qubit state equation
  • Bloch vector coordinates
  • Measurement probabilities

6. Quantum Gates

  • I, X, Y, Z
  • Hadamard (H)
  • Phase gates: S, S†, T, T†

Gate actions are animated, and the trajectory of the state is traced on the Bloch sphere.

7. Rotation Gates

Apply continuous rotations using Rx, Ry, and Rz with a chosen angle.

8. Controls

  • Reset – return to |0⟩
  • Undo – revert last operation
  • Redo – reapply operation

9. Learning Tips

  • Compare H and Ry(90°)
  • Apply Z after X and observe phase
  • Study trajectories for gate sequences
